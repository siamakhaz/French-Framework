# TEFAQ Writing — Section B Overview

This section adds a structured preparation path for the **TEFAQ writing exam, Section B**.

## Focus
- formal letter format
- minimum 200 words
- clear structure
- present tense by default
- memorization-friendly sentence blocks

## Core method
The writing approach here is based on:
- a fixed formal-letter template
- reusable phrase blocks
- simple arguments and examples
- consistent exam-style structure

## Recommended workflow
1. Read the topic.
2. Choose your position quickly.
3. Fill the template with simple arguments.
4. Reuse memorized phrases from the cheat sheet.
5. Check agreement, verb endings, and fixed expressions.

## What matters most
- structure
- grammatical accuracy
- coherence
- useful connectors
- formal tone
